ColorFriendly

A Chrome extension that makes it easier for color blind people to delineate different elements on a webpage

Download .zip Download .tar.gz View on GitHub

Welcome to ColorFriendly

ColorFriendly is a Chrome extension that makes it easier for people who are red-blue and blue-green color blind to discern the difference between similarly-colored elements on webpages. User settings are persistent, and the user interface is as simple and as direct as possible to minimize the amount of post-installation management needed. Set it up once and you can forget about it.

Extension Installation

To install the extension, simply download the crx file from the master branch (located here; click "View Raw" to download the file), navigate to the Extensions page in Chrome, and drag and drop the crx from your Downloads folder to the extensions page. Click "Add Extension" to finish installation.

Extension Use

After ColorFriendly is installed, it is super easy to use. Click on the toolbar icon to choose which kind of adjustment you want (red-blue or blue-green) and whether you want the extension to change colors at all. This second option is particularly handy if you want to disable the color changing functionality without uninstalling the extension or digging through the Chrome extension settings page. And that's it! That's all that's required to use ColorFriendly.
